# Mer

A squad of robotic rovers are to be landed by NASA on a plateau on Mars

## Installation

NOTE: Was developed with ruby `v.2.2.3`

Execute:

    $ bundle


## Test

```
bundle

rake
```

## Usage

```
 ./bin/discover mission.txt 2>/dev/null
```
